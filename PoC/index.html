PRAI PoC

Welcome to the PRAI Proof of Concept

This is a simple web application that allows you to generate a final draft based on a source file and a style reference.

PRAI will generate a document based on the content of the source file and the writing style/tone of the style reference.

The title you provide will specify what type of document you want to generate as well as the stance you want to take on the topic.

Upload your source file (PDF/DOCX), specify the topic, length, and upload a style reference (PDF/DOCX) to generate the final draft.

Please name the files appropriately so that they are easy for the system to identify.

The source file should contain the event you're writing about or information about the topic.

The style reference should contain the writing style or tone you want to emulate. This could be a previous article, a blog post, or even your brand guidelines.

Click the "Generate" button to create the final draft.
