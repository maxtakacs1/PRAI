PRAI PoC

Welcome to the PRAI Proof of Concept

This is a simple web application that allows you to generate a final draft based on a source file and a style file.

PRAI will generate a document based on the content of the source file and the writing style of the style file.

The title you provide will specify what type of document you want to generate as well as the stance you want to take on the topic.

Upload your source file (PDF/DOCX), specify the topic, length, and upload a style file (PDF/DOCX) to generate the final draft.

Click the "Generate" button to create the final draft.
